## Planned Hardware
- WS2812B LED strip (5m, 144LED/m)
- 2×3 ft acrylic sheet
- One‑way mirror film
- 5V 30A power supply
- ESP32 microcontroller
- PLA filament for 3D printing
- Single‑core + stranded wire
- Heat shrink + connectors
